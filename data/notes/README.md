# Tourism Website

A single-page tourism website featuring a centered image carousel showcasing various travel destinations. Built with pure HTML, CSS, and JavaScript - no frameworks required.

## Features

- **Image Carousel**: Auto-rotating carousel with 4 beautiful destination images
- **Manual Navigation**: Previous/next buttons (appear on hover) and dot indicators
- **Fully Responsive Design**: Optimized for mobile phones, tablets, and desktop computers
- **Mobile Menu**: Collapsible navigation menu for mobile devices
- **Modern UI**: Clean design with smooth transitions and animations
- **Touch-Friendly**: Large tap targets for better mobile interaction
- **Pure HTML/CSS/JS**: No frameworks or build tools required

## Structure

- **Header**: Sticky header with brand logo, responsive navigation, and mobile menu
- **Carousel**: Centered image slideshow with responsive sizing and captions
- **Footer**: Copyright information with responsive text sizing

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `script.js` - Carousel and menu functionality
